# SymfonyKitapSatis sitesidir . 

Kullanıcı ürün ekler(admin onaylı ) , ürün ekler. 

Metehan Kafadar Tarafından yazılmıştır. MySQL db sinin sıfırlanması ve yeni ürünler eklenmesi gerekmektedir. 

Django projesinin içerisinde kullandığım template'i kullandım. Ona entegre yapılıp gerekli bağlantılardan sonra ürün eklemeleri yapıldığında sorunsuz çalışacaktır.

Satış yapıyor ancak gerekli kredi kartı API vs olmadığı için bir sorgulama yapılmadı. Sadece bilgileri admine gönderiyor ve onay durumunda stoktan düşüş gerçekleşiyor.
